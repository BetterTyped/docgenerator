# Docgen

> Docgen is a tool for generating documentation from TypeScript source code. It is designed to support additional
> semi-automated way of creating documentation by importing parts of the generated output for amazing DX.

:::note Work in progress

Please note that this documentation is a work in progress and we will be adding more modules soon. Until this time you
can check the [source code](https://github.com/BetterTyped/docsgen) for more information. In case of any questions
[contact us](mailto:maciekpyrc@gmail.com).

:::
